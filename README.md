# PiriCase1

#Case
1 - Array içindeki sayılara sırayla bakıldığında ilk duplicate olarak karşılaşan sayıyı bul.
Eğer duplicate yoksa -1 sonucunu dön.
Örnek: numbers = [5, 3, 4, 2, 4, 5] sonuç = 4,
Örnek: numbers = [3, 3] sonuç = 3,
Örnek: numbers = [2, 8 , 3, 4] sonuç = -1

#Çözüm

1. Arraydeki elemanlar üzerinde teker teker inceleme yapılmalıdır.
2. Arraydeki ilk tespit edilen dublicate elemanda looptan çıkış sağlanmalıdır.
3. Tespit edilen eleman dönülmelidir.
4. Tespit edilen bir eleman yoksa -1 dönülmelidir.
